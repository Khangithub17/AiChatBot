AIChatbot is a responsive web-based chatbot interface that interacts with users and fetches responses from a language model API. Built with HTML, CSS, and JavaScript, this project demonstrates real-time AI-powered chat interactions using asynchronous API calls.

Features
User-Friendly Interface: Clean and simple design with a chat-style layout.
Asynchronous API Calls: Sends user queries to an AI model API and displays real-time responses.
Loading Indicator: Shows a loading icon while awaiting API responses.
Error Handling: Provides user-friendly error messages if there are issues fetching data.
Technologies Used
HTML, CSS, JavaScript
Fetch API for asynchronous communication
Git for version control
How to Use
Clone the repository.
Insert your own API key in the designated area of the JavaScript file.
Open the index.html file in your browser and start chatting!
Setup
Run git clone https://github.com/your-username/aichatbot.git
Open the project in VS Code and replace the placeholder API key in the api_Url variable.
Enjoy real-time chat with your AI bot!
Feel free to contribute or suggest improvements!
